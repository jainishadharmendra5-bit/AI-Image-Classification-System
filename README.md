# AI-Image-Classification-System
AI Image Classification Chatbot built with PyTorch, FastAPI, and Streamlit. It uses a custom CNN trained on CIFAR-10 to classify images, return confidence scores, and provide chatbot-style responses with facts. Supports real-time image upload and interactive AI explanations.
